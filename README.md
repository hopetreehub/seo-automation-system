# 🚀 자동 블로그 포스팅 시스템 v2.0

**3개 블로그 완전 자동화 운영 시스템 - 하루 6회 × 주 7일 = 주 42회 자동 포스팅**

## ✨ 시스템 개요

WordPress 기반 3개 전문 블로그를 24시간 완전 자동으로 운영하는 시스템입니다. AI가 키워드 분석부터 글 작성, 예약 게시까지 모든 과정을 자동화합니다.

### 🎯 지원 블로그

| 블로그 | 전문분야 | 기본 스케줄 | 특징 |
|--------|----------|-------------|------|
| 💰 **INBECS 생활경제** | 정부혜택, 지원금, 세금정보 | 하루 6회 | 시점별 필수 생활정보 |
| 📊 **INBECS 비즈니스** | 투자분석, 경제전망, 시장동향 | 하루 6회 | 전문적 경제 분석 |
| ✨ **INNERSPELL** | 자기계발, 심리학, 라이프스타일 | 하루 6회 | 개인 성장 콘텐츠 |

## ⏰ 자동 포스팅 스케줄

### 📅 기본 스케줄 (하루 6회)
```
06:00 - 새벽 포스팅 (긴급/중요 정보)
09:00 - 아침 포스팅 (출근길 정보)
12:00 - 점심 포스팅 (점심시간 읽을거리)
18:00 - 저녁 포스팅 (퇴근길 정보)
21:00 - 밤 포스팅 (여가시간 콘텐츠)
00:00 - 심야 포스팅 (다음날 대비 정보)
```

### 🗓️ 주간 커스터마이징
- **월요일**: 6회 (기본)
- **화요일**: 10회 (많이)
- **수요일**: 0회 (쉬는 날)
- **목요일**: 4회 (적게)
- **금요일**: 8회 (많이)
- **토요일**: 3회 (적게)
- **일요일**: 5회 (보통)

**총 주간 포스팅**: 사용자 설정에 따라 0~168회 (24시간×7일)

## 🎛️ 시스템 기능

### 🤖 완전 자동화 기능
- ✅ **키워드 자동 분석**: AI가 트렌드/시점별/긴급 키워드 자동 선택
- ✅ **제목 자동 생성**: SEO 최적화된 클릭율 높은 제목 10개 생성
- ✅ **인간형 글 작성**: AI 티 안 나는 자연스러운 글쓰기
- ✅ **WordPress 자동 게시**: 메타데이터 포함 완전 자동 업로드
- ✅ **스케줄 관리**: 정확한 시간 기반 자동 실행

### 📊 스마트 기능
- 🧠 **AI 스마트 선택**: 시간대/요일별 최적 키워드 자동 선택
- 🚨 **긴급정보 우선**: 마감 임박 지원금 등 우선 처리
- 📈 **실시간 통계**: 포스팅 현황 및 성과 추적
- 💾 **기록 관리**: 포스팅 히스토리 자동 저장

## 🔧 설정 옵션

### 자동화 레벨
- **수동 실행**: 원클릭 포스팅
- **예약 실행**: 지정 시간 포스팅
- **매일 자동**: 하루 6회 자동 포스팅
- **스마트 자동**: AI가 최적 시점 판단

### 우선순위 설정
- 🚨 **긴급정보 우선**: 긴급 → 시점별 → 일반
- ⏰ **시점별 우선**: 시점별 → 정부혜택 → 트렌드
- 🔄 **균형 순환**: 모든 카테고리 고르게
- 🤖 **AI 스마트**: 시간대별 최적 선택

## 📋 사용 가이드

### 1단계: 블로그 선택
```
💰 INBECS 생활경제 클릭
   ↓
자동화 설정 패널 활성화
```

### 2단계: 자동 설정
```
자동 포스팅: "매일 자동 (하루 6회)" 선택
우선순위: "긴급정보 우선" 추천
글 스타일: "실용정보형" 자동 설정
```

### 3단계: 주간 스케줄 (선택사항)
```
"주간 스케줄 직접 설정" 선택
   ↓
요일별 시간 커스터마이징
   ↓
"주간 스케줄 저장"
```

### 4단계: 시스템 시작
```
"자동 포스팅 활성화" 클릭
   ↓
24시간 완전 자동 운영 시작!
```
